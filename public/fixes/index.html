VSCode C++ 函数太长参数太多看不清赋值给了哪个参数

2026 年 6 月 1 日

下载插件Settings Cycler ctrl+shift+p 打开键盘快捷方式

## keybindings.json
// 参数提示
    {
    "key": "ctrl+shift+j",
    "command": "settings.cycle",
    "args": {
        "id": "toggleCppParameterHints",
        "values": [
            {
                "C_Cpp.inlayHints.parameterNames.enabled": true
            },
            {
                "C_Cpp.inlayHints.parameterNames.enabled": false
            }
        ]
    },
    "when": "editorTextFocus && (editorLangId == c || editorLangId == cpp)"
    }

然后按 ctrl+shift+j 就可以切换了 如果报错没效果检查一下项目目录下 .vscode\settings.json 看是不是设置 inlayHints 了，会覆盖
